# PlasmaPal
A web based application which focuses on easing the process of plasma donation thereby encouraging more people to donate plasma and save lives.
## Features of the application :
* Check eligibilty for plasmadonation
* Look for nearby donation centres
* Book a slot for donation
* Incase a donor wants to report some uneasiness:
    * Chat with a doctor
    * Video consult a doctor
* Post a plasma request
* Search for plasma matches

### Tech Stack used :
      MongoDB - Database used. Will change in the future when API used instead.
      React - Frontend JavaScript library.
      Express -Server/routing API for web app.
      Node - Backend JS runtime.
      WebRTC - Video consultation.
      Socket.io - Chat with doctor.
      Multer - Image storage for covid report.
